# 🎮 Tic-Tac-Toe Game

A modern, responsive Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against a friend or challenge the computer AI!

## ✨ Features

- **Two Game Modes**
  - 👥 **2-Player Mode** – Take turns with a friend on the same device
  - 🤖 **vs Computer Mode** – Challenge our smart AI opponent

- **Score Tracking** – Automatically tracks wins for X, O, and draws across multiple games

- **Visual Polish**
  - Smooth animations and hover effects
  - Winning cells highlight in green with a pulse animation
  - Gradient backgrounds and modern UI design
  - Fully responsive – works on desktop, tablet, and mobile

- **Smart AI** (in vs Computer mode)
  - Tries to win when possible
  - Blocks player from winning
  - Prioritizes center and corners
  - Falls back to random moves

## 🚀 How to Play

1. **Open the game** – Double-click the `tictactoe.html` file in your browser
2. **Choose game mode** – Click "2 Players" or "vs Computer"
3. **Make your move** – Click any empty cell to place your X or O
4. **Win the game** – Get 3 in a row (horizontally, vertically, or diagonally)
5. **Track scores** – Scores persist until you reset them
6. **Play again** – Click "New Game" to start a fresh round

## 🎯 Game Rules

- Players take turns placing X and O on a 3×3 grid
- First to get 3 in a row wins
- If all 9 cells are filled with no winner, it's a draw
- In 2-player mode, X always goes first
- In vs Computer mode, you play as X, computer plays as O

## 🛠️ Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling, animations, and responsive design
- **JavaScript (ES6+)** – Game logic, AI, and DOM manipulation
